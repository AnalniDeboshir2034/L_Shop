# 🚀 ДЕПЛОЙ НА VERCEL - БЫСТРАЯ ИНСТРУКЦИЯ

## ✅ Всё готово! Просто выполни эти шаги:

---

## Шаг 1: Запуш изменения

```bash
git push
```

---

## Шаг 2: Зайди на Vercel

1. Перейди на https://vercel.com/dashboard
2. Найди свой проект **fl**
3. Если проекта нет - импортируй из GitHub

---

## Шаг 3: Проверь деплой

После пуша Vercel автоматически задеплоит изменения.

**Жди 1-2 минуты** пока идет деплой.

---

## Шаг 4: Проверь сайт

Открой https://fl-ynid.vercel.app/

### Что должно работать:

✅ **Товары** - должны отображаться 3 товара:
- MaFinBuSi Runner X (7990 ₽)
- MaFinBuSi Urban Hoodie (4990 ₽)  
- Трусы Calin Kelai (232 ₽)

✅ **Регистрация** - работает через `/api/auth/register`

✅ **Вход** - работает через `/api/auth/login`

✅ **Корзина** - добавление товаров работает

✅ **Заказы** - оформление заказа работает

---

## 🔧 Как это работает теперь:

### Автоматическая инициализация
При первом посещении каталога:
1. Сайт проверяет есть ли товары
2. Если товаров нет → вызывает `/api/init`
3. Данные создаются автоматически
4. Товары отображаются

### Хранение данных
- **Без BLOB_READ_WRITE_TOKEN**: данные хранятся во временном хранилище Vercel
- **С BLOB_READ_WRITE_TOKEN**: данные хранятся в Vercel Blob (постоянно)

---

## 📦 Опционально: Vercel Blob Storage

Для постоянного хранения данных (10 GB бесплатно):

1. Vercel Dashboard → Project → **Storage**
2. **Connect Database** → **Vercel Blob**
3. Создать Blob store
4. `BLOB_READ_WRITE_TOKEN` добавится автоматически

**Без Blob Storage тоже работает!** Данные будут инициализироваться при каждом деплое.

---

## 🆘 Если товары не отображаются

1. Открой консоль браузера (F12)
2. Обнови страницу
3. Посмотри логи - должно быть "No products found, initializing..."
4. Подожди 2-3 секунды

Или вручную вызови инициализацию:
```bash
curl -X POST https://fl-ynid.vercel.app/api/init
```
